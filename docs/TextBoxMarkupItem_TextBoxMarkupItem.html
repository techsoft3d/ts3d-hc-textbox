The background color of the textbox

The color of the circle that is rendered at the insertion point
circleRadius number 4

maxWidth number 300

The maximum width of the textbox
fixed boolean false

If true, the textbox will be fixed in position
extraDiv string null

Extra div text to be added to the textbox
uniqueid guid null

Id of Markup Element
userdata object null

User data to be stored with the markup element
checkVisibility boolean false

If true, the textbox will be hidden if the insertion point is not visible
showLeaderLine boolean true

If true, the textbox will have a leader line
